# Date Time Range Picker

Custom Date Time Range Picker built using React and Storybook.

## Features
- Keyboard-first accessibility
- Timezone selection
- DST-safe formatting using Intl.DateTimeFormat
- Min/Max constraints
- Presets (Today, Last 7 Days)
- Storybook documentation with edge cases

## How to run
npm install  
npm run storybook
